## 📖 项目简介

**全自动AI游戏帝国**是一个将古代唐朝“三省六部制”的分权、制衡与专业化思想，应用于现代AI多智能体协作的**游戏开发自动化框架**。

它通过模拟一个由 **12个专职AI Agent** 组成的“赛博朝廷”，将你（皇帝/项目总监）的模糊需求，自动拆解、审核、派发、执行，最终产出可运行的代码、资产和文档，实现 **L5级全自动游戏开发**。

---

## 🧠 核心理念

| 古代官制 | 现代职能 | 在本框架中的AI Agent角色 |
| :--- | :--- | :--- |
| **皇帝** | 项目总监 | **你**：提出愿景，做出关键决策。 |
| **中书省** | 创意与规划 | 负责需求分析、方案设计与任务拆解。 |
| **门下省** | 审核与风控 | 负责审核方案可行性，不合格则“封驳”打回。 |
| **尚书省** | 调度与协调 | 负责将任务派发给专业部门，追踪进度。 |
| **六部** | 专项执行 | **兵部**(代码)、**户部**(数值)、**礼部**(文档)、**刑部**(测试)、**工部**(资产/部署)、**吏部**(Agent管理)。 |
